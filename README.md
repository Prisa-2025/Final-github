# Simple Interest Calculator

## Project Description
This project is a simple Bash-based calculator that computes **simple interest** based on user input.  
The user is prompted to enter the principal amount, the rate of interest, and the time period, and the script calculates the simple interest accordingly.

## How It Works
The calculator uses the simple interest formula:

Simple Interest = (Principal × Rate × Time) / 100

## Input Fields
- **Principal**: The initial amount of money
- **Rate of Interest**: The interest rate (in percentage)
- **Time Period**: The duration for which the interest is calculated

## Technology Used
- Bash Shell Script

## Usage
Run the `simple-interest.sh` script in a terminal and follow the on-screen prompts to enter the required values.
